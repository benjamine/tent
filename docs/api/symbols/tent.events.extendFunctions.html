  • <static> tent.events.extendFunctions.bind(eventType, data, handler)
    Binds functions to an event (or events) 
    		// bind by name
		obj.bind('click', function(e) {
			alert(this.id+' clicked at '+e.data.position);
		};

		// bind with a map
		obj.bind({
			activate: function(e) {
				console.log('activated');
		}, desactivate: function(e) {
				console.log('deactivated');
		});

		// bind passing data
		obj.bind('load', { context: this.title }, function(e) {
			alert(e.data.context+' clicked at '+e.data.position);
		};
    Parameters:
    {String} eventType
    {Object} data Optional
    data to include in eventData, if an order field is included it's used to sort handler invocations
    {Object|function()} handler
    Returns:
    the context object (this) for chaining
