<static> {Boolean} tent.changes.track(obj, options)
Activates Change Tracking on a Javascript Object or Array. Optionally binds change handlers. This function is idempotent.

The attached tent.changes.Observable is stored on the obj.__observable__ property.

Parameters:
{Object|Array} obj
an Object or Array to track.
{Object} options Optional
tracking options
{tent.changes.Observable} options.observable Optional
an tent.changes.Observable to attach (otherwise a new one is created)
{Boolean} options.remove Optional
if true, indicates that specified bindings must be removed
{Boolean} options.removeAll Optional
if true, indicates that all bindings, even tent.changes.Observable instances must be removed
{Object} options.interceptOptions Optional
options to use when calling tent.changes.Observable.interceptProperties or tent.changes.Observable.interceptArrayModifiers
{Object[]} options.bindings Optional
bindings to add (or remove, see options.remove), eg: { eventType: tent.changes.EventTypes.CHANGED, handler: function(change){alert(change+'');} };
{Boolean} options.log Optional
if true log handlier is included, see tent.changes.LogHandler
{Boolean} options.reverseProperties Optional
if true reverse properties handling is included, see tent.changes.reverseProperties.getHandler
{Boolean} options.deep Optional
if true recursively traverse Object properties and Array items
{Boolean} options.live Optional
if true combined with options.deep, when new Objects or Arrays are added/linked to tracked ones they get tracked too
{function()} options.propertyFilter Optional
if true combined with options.deep, when traversing Object properties, properties that doesn't satisfy this condition are ignored
Returns:
{Boolean} false if no action has been performed (ie: the object was already tracked)
