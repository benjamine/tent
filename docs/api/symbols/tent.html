<static> {tent.Namespace} tent.declare(parent, path, expander)
Declares Namespaces and add members to them (Namespace paths are unique, ie: the same Namespace object is returned for the same path)

Parameters can be any sequence of:

  • tent.Namespace: changes the current Namespace
  • {string} path: changes current Namespace to a subnamespace of the current Namespace (creating Namespaces if needed)
  • {function|Object} expander: a function that adds members to the current Namespace or an Object wich properties are copied to the current Namespace

Parameters:
{tent.Namespace} parent Optional
parent Namespace
{String} path Optional
Namespace path, eg: 'web.ajax.json.parsing'
{function|Object} expander Optional
a function that receives the created Namespace and adds members (eg: classes, constants, functions) or an Object wich properties are copied to the current Namespace
Returns:
{tent.Namespace} the last current Namespace
