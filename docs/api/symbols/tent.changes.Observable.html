    Intercepts all array modifying functions on #subject (if #subject is an Array)

    The only modifications that are not intercepted are:

    • setting by index, ie: array[index]=value, use array.set(index,value) instead.
    • setting array length property, ie: array.length = newlength, use array.setLength(newLength) instead.
