    Clones objects This is useful in browsers that only support property change tracking on DOM objects (eg: IE8)

    {Boolean} options.deep Optional
    clone deeply, traverse through object properties?
    {Boolean} options.clonePrivates Optional
    clone private members (properties starting with _)
    {Boolean} options.skipPrivates Optional
    skip (don't copy) private members (properties starting with _)
    {Boolean} options.attachedObjectsIds Optional
    if true objects attached to a tent.entities.Context are replaced by their ids (useful for persistence serialization)
    {Boolean} options.attachedObjects Optional
    if true objects attached to a tent.entities.Context are cloned too, by default is true (if false deep cloning stops at attached objects)
    {Boolean} options.ignoreCircularReferences Optional
    if cloning deeply, ignore circular references (otherwise error is thrown)
    {Boolean} options.onlyForTracking Optional
    only clone if the current browser only support tracking on DOM objects (eg: IE8), otherwise original obj is returned
    Returns:
    {Object} a clone of obj (or obj if no cloning was performed)

  • <static> {tent.Namespace} tent.declare(parent, path, expander)
    Declares Namespaces and add members to them (Namespace paths are unique, ie: the same Namespace object is returned for the same path)

    Parameters can be any sequence of:

    • tent.Namespace: changes the current Namespace
    • {string} path: changes current Namespace to a subnamespace of the current Namespace (creating Namespaces if needed)
    • {function()|Object} expander: a function that adds members to the current Namespace or an Object wich properties are copied to the current Namespace

    Parameters:
    {tent.Namespace} parent Optional
    parent Namespace
    {String} path Optional
    Namespace path, eg: 'web.ajax.json.parsing'
    {function()|Object} expander Optional
    a function that receives the created Namespace and adds members (eg: classes, constants, functions) or an Object wich properties are copied to the current Namespace
    Returns:
    {tent.Namespace} the last current Namespace

  • <static> tent.pset(subject, propertyName, value, skipInterceptors)
    Sets an object property value. If subject is an observed object, it ensures that its tent.changes.Observable is notified. Its useful when no property intercept mode is supported {link tent.changes.PropertyInterceptModes}.
    Defined in: changes.js.
